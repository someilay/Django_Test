## Requirements:

* Python 3.9+
* Django 4.2+

## Installation & Run

* `cd git_dir`
* `python3 -m venv venv`
* `pip install -r requirements.txt`
* `cd test_site`
* `python3 manage.py runserver`

## Prerequisites

* `test_site\example_db` - database with menu examples, one linux structure like, another - light weighted
* Admin account: admin; Password: admin
